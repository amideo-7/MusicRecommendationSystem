Intro.

Discovering new music has never been easier. Powered by cutting-edge machine learning technology, Music Recommendation System is here to elevate your musical experience to new heights. prepare to explore a world of melodies you never knew existed. From hidden gems to undiscovered artists, let the system be your trusted companion on a journey of auditory exploration.

Technology.

This Music Recommendation System uses machine learning based Collaborative Filter, that predicts what a user like based on what other users, that are similar to that particular user, have liked. Collaborative Filter work with an interaction matrix , also called rating matrix. The aim of this algorithm is to learn a function that can predict if a user will benefit from an item-meaning the user will likely listen this item.
